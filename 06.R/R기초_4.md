# R 기초 문법 (4) 연산자

## 비교연산자

* `==`
* `!=`
* `>`  `<`  `>=`  `<=`

* 벡터와 비교연산자

  ```R
  x <- c(0,25,50,75,100)
  x == 50   # FALSE FALSE  TRUE FALSE FALSE
  ```

  * 벡터와 단일값 비교 시, 단일값이 벡터의 값들과 각각 대응됨
  * 위 예시에서 50과 비교연산 시 `c(50, 50, 50, 50, 50)` 과 비교하는 것과 같음



## 산술연산자

### 나눗셈 관련

* `/` :  일반 나눗셈 연산 (소수 자리까지 연산)
* `%%` :  나머지 (정수형)
* `%/%` :  몫 (정수형)
